# PHÂN CÔNG CÔNG VIỆC NHÓM

## Thành viên 1: Minh Quân
- Xây dựng Server & API (Backend): Thiết lập server (NodeJS/Express) hoặc JSON Server để xử lý dữ liệu.
- Quản lý Database: Thiết kế Schema, lưu trữ và truy xuất dữ liệu sản phẩm, đơn hàng.
- Chức năng Xác thực (Authentication): Xây dựng chức năng Đăng nhập, Đăng ký, phân quyền User/Admin.
Xử lý Logic nghiệp vụ:
- Logic thêm/xóa/sửa sản phẩm ở trang Admin (CRUD).
- Logic tính toán giỏ hàng, đặt hàng và thanh toán.
- Tích hợp API: Kết nối dữ liệu từ Server về giao diện Frontend (Fetch data).
- Lập trình CSS/SASS, Dark Mode và Responsive.
## Thành viên 2 Kiên Hiền
- Xây dựng giao diện Trang chủ, Chi tiết sản phẩm, Giỏ hàng.
- Xây dựng giao diện Trang quản trị Admin.
- Logic tính toán giỏ hàng, đặt hàng và thanh toán.
- Xây dựng giao diện chức năng Tìm kiếm và Lọc sản phẩm.
